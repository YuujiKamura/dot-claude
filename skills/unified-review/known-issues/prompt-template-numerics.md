# AI Prompt Template Known Issue Patterns

## JSONテンプレートに具体的な数値を入れるな
- **検出条件:** JSONテンプレート内に `1.0`, `0.5` 等の具体的な例値がプレースホルダとして使われている
- **問題:** AIが画像観察を飛ばしてテンプレートの例値をそのままコピーする。全推定値が例値と同じになる
- **正しい対処:** 数値プレースホルダは `0`、文字列プレースホルダは `"?"`。具体的な範囲はプロンプトtext内の別セクション(rangeGuide等)に記述
- **発見日:** 2026-02
- **発生元:** tonsuu-checker プロンプト設計の試行錯誤
