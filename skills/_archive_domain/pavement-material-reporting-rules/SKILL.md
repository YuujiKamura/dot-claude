## 1. Pavement Material Reporting Rules

Decision criteria for material accounting in road repair. Certain categories like district marking (区画線) allow 'planned' values to be used as 'actual' values in reports because they typically match, whereas asphalt mixtures (合材) require precise 'actual' values derived from temperature management charts.

### Steps

1. Identify the source for 'Actual' values: use Temperature Management Charts for asphalt and Material Verification PDFs for planned quantities
2. For specific work types like district marking, copy planned values to actual fields when direct measurements are not provided
3. Exclude material categories (e.g., PK-3) from the aggregate sheet if the specific project scope does not use them
