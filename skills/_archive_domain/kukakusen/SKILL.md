# 区画線

会話数: 1 | パターン数: 4

## 1. 書類間の整合性確認（不整合検知）

複数の工事書類（計画まとめ、面積展開図、切削計算書など）を比較し、数値の不一致や転記ミスを出発点として特定する。

### 手順

1. 対象となる複数のファイル（Excel、CSV等）を読み込む
2. 面積、数量、材料名などの主要項目を抽出する
3. 書類間で値が一致しているか突合する
4. 不整合がある場合、どの書類が根拠として妥当か（または単なる入力ミスか）を分析する

出現頻度: 3回

## 2. 舗装面積算出の優先順位判断

舗装面積の根拠資料が複数ある場合、算出精度の高い書類を優先的に参照する。

### 手順

1. 「面積展開図」と「切削計算書」を比較する
2. 「面積展開図」の数値を正解の基準とする（切削計算書は丸め処理の関係で誤差が出やすいため）
3. 合計値だけでなく、算出過程の数値も整合しているか確認する

出現頻度: 2回

## 3. 専門用語・材料名の事実確認（創作禁止）

「メガラックス」などの特定のブランド名や専門用語に対し、推測で回答せず、ファイル内容や客観的根拠に基づいて説明する。

### 手順

1. 未知の用語が現れた場合、まずファイル内の該当箇所を検索する
2. ファイルに定義がない場合は、推測せず「不明」であることを伝えるか、外部知識として確認をとる
3. 材料の区分（J17：通常、J20：高輝度など）がファイル上の定義と一致しているか厳密に確認する

出現頻度: 2回

## 4. 日本語Windows環境向けCSV/Excel操作

日本の工事現場で一般的に使用されるExcel環境（Shift-JIS/cp932）に対応したデータ処理を行う。

### 手順

1. CSVファイルを扱う際はエンコーディングを cp932 (Shift-JIS) に設定する
2. Excelの数式（SUM等）を設定する際、項目の内容を精査して範囲（J17, J20, J21など）を動的に決定する
3. 集計行や合計列が、個別の材料項目の単純合算になっているか検証する

出現頻度: 2回
