## 新規シート作成手順

1. **12月26日シートをコピー**（お手本）
2. シート名を日付に変更（例: `1月15日`）
3. 以下のセルを更新：
   - C5: 工事場所（地域情報）
   - B8: 日付
   - C8: 開始時刻
   - E8: 終了時刻
   - F8: 休憩（分）
   - H8: 昼間人数
   - I8: 夜間人数（該当する場合）
   - K8, M8: 警備員名
4. 行9以降は必要に応じて入力
5. **バリデーションを実行**

## バリデーションチェック項目

### 1. データ型チェック
- B列: datetime型であること
- C,E列: time型であること
- F,H,I列: int型であること（文字列不可）
- G,J列: 数式であること（値の直接入力不可）

### 2. 数式チェック
- G列: `=VALUE(E-C)*24-F/60` パターン
- J列: `=G*H` または `=G*I` パターン
- G18: `=AVERAGE(G8:G17)`
- H18: `=SUM(H8:H17)`
- J18: `=SUM(J8:J17)`

### 3. 書式チェック
- B列: 日付書式
- C,E列: 時刻書式 `h:mm`
- H,I列: 人数書式 `0"人"`
- G列: 小数書式 `0.0`

### 4. 構造チェック
- 結合セルが正しいこと
- ラベルセルが変更されていないこと

### 5. テキスト配置チェック
- 同一列内で配置（左/中央/右）が異なるセルを検出
- 異常なセルがあれば警告

## テキスト配置バリデーション

```python
from openpyxl import load_workbook
from collections import Counter

def check_alignment(excel_path, sheet_name, check_columns=['B', 'C', 'E', 'F', 'G', 'H', 'K', 'M'], data_rows=range(8, 18)):
    """テキスト配置の一貫性をチェック

    Args:
        excel_path: Excelファイルパス
        sheet_name: シート名
        check_columns: チェック対象の列
        data_rows: データ行の範囲

    Returns:
        警告リスト
    """
    wb = load_workbook(excel_path)
    ws = wb[sheet_name]
    warnings = []

    for col in check_columns:
        alignments = []
        for row in data_rows:
            cell = ws[f'{col}{row}']
            if cell.value is not None:
                h_align = cell.alignment.horizontal or 'general'
                v_align = cell.alignment.vertical or 'bottom'
                alignments.append((row, h_align, v_align))

        if len(alignments) >= 2:
            # 最頻値を取得
            h_counts = Counter([a[1] for a in alignments])
            v_counts = Counter([a[2] for a in alignments])
            common_h = h_counts.most_common(1)[0][0]
            common_v = v_counts.most_common(1)[0][0]

            # 最頻値と異なるセルを警告
            for row, h, v in alignments:
                if h != common_h:
                    warnings.append(f'{col}{row}: 水平配置が異常 ({h}) - 他は {common_h}')
                if v != common_v:
                    warnings.append(f'{col}{row}: 垂直配置が異常 ({v}) - 他は {common_v}')

    wb.close()
    return warnings

# 使用例
warnings = check_alignment(
    '4交通誘導員配置実績集計表、東区2工区.xlsx',
    '12月26日'
)
for w in warnings:
    print(f'警告: {w}')
```

## PDF出力と検証

### 番号体系

提出書類番号: **41** （交通誘導員配置実績集計表）

ファイル名形式: `41_交通誘導員配置実績集計表_MMDD.pdf`

### Excel→PDF変換

```python
import win32com.client
import os
import re

def export_traffic_guard_pdf(excel_path, sheet_name, output_dir=None):
    """交通誘導員配置実績集計表をPDFに変換

    Args:
        excel_path: Excelファイルパス
        sheet_name: シート名（例: '12月26日'）
        output_dir: 出力先フォルダ（省略時はExcelと同じ場所）

    Returns:
        出力したPDFのパス
    """
    # シート名から日付部分を抽出（例: '12月26日' → '1226'）
    match = re.search(r'(\d+)月(\d+)日', sheet_name)
    if match:
        date_str = f"{int(match.group(1)):02d}{int(match.group(2)):02d}"
    else:
        date_str = sheet_name.replace('月', '').replace('日', '')

    # 番号体系に従ったファイル名（日付付き）
    pdf_name = f'41_交通誘導員配置実績集計表_{date_str}.pdf'

    if output_dir is None:
        output_dir = os.path.dirname(excel_path)
    pdf_path = os.path.join(output_dir, pdf_name)

    excel = win32com.client.Dispatch('Excel.Application')
    excel.Visible = False
    excel.DisplayAlerts = False

    try:
        wb = excel.Workbooks.Open(os.path.abspath(excel_path))
        ws = wb.Worksheets(sheet_name)
        ws.ExportAsFixedFormat(0, os.path.abspath(pdf_path))
        wb.Close(False)
        return pdf_path
    finally:
        excel.Quit()

# 例: 12月26日シートをPDF化（請求時フォルダに出力）
export_traffic_guard_pdf(
    '4交通誘導員配置実績集計表、東区2工区.xlsx',
    '12月26日',
    '請求時/'
)
# → 請求時/41_交通誘導員配置実績集計表_1226.pdf
```

### PDF検証チェック項目

| 項目 | 検証内容 |
|------|---------|
| 日付 | シート名と一致するか |
| 時間帯 | 開始・終了時刻が正しいか |
| A時間 | (終了-開始)-休憩/60 と一致するか |
| B人数 | 警備員名の人数と一致するか |
| 合計時間 | A時間×B人数と一致するか |
| 警備員名 | 伝票と照合（OCR誤読注意） |
| 工事場所 | 当該工区の地域情報か |

### 視覚的な違和感チェック

**PDFを見たとき「なんか1つだけ違う」に気づく**

| 違和感 | 例 |
|--------|-----|
| 配置ずれ | 1セルだけ左寄せ、他は中央 |
| 位置ずれ | 1行だけ微妙に上下にずれている |
| フォント | 1セルだけ太字/斜体/サイズ違い |
| 罫線 | 1箇所だけ線が太い/細い/欠けている |
| 色 | 1セルだけ背景色や文字色が違う |

**チェック方法:**
1. PDF全体を俯瞰して「揃っているか」を確認
2. 各列を縦に見て配置の統一感を確認
3. 「1つだけ違う」ものがあれば警告

## ガードマン集計の定番フロー（南千反畑運用）

### 1. 雛形を複製して案件フォルダへ配置
1. 既存の `4交通誘導員配置実績集計表` をコピー。
2. 新案件名でリネームして `19実施数量` へ配置。
3. `template-copy` スキル手順で元/先を記録。

### 2. 伝票を抽出（photo-tagger）
1. 入力: `ガードマン伝票.pdf`
2. 実行: `--voucher-type guardman`
3. 出力: `ガードマン伝票_guardman.xlsx`（中間JSONも保存）

### 3. 誘導員集計表へ転記
1. 運用シートを1枚に決める（`Link` は残す）。
2. `guardman.xlsx` の `work_date/start_time/end_time/break_minutes/worker_names/worker_count` を `B8:M17` へ転記。
3. 工事名は案件ルート名に合わせて `C4` を更新。
4. 夜間判定:
   - `start>=18:00` または `end<=06:00` または `end<start` を夜間扱い
   - 夜間は `I` 列、昼間は `H` 列に人数を入れる
5. 氏名は `worker_names` を `|` で分割して `K` / `M` に振り分け（改行区切り）。

### 4. 数式を統一
- `G8:G17`: `=MOD(E?-C?,1)*24-F?/60`（夜跨ぎ対応）
- `J8:J17`: `=G?*IF(I?>0,I?,H?)`
- 集計行:
  - `G18 =AVERAGE(G8:G17)`
  - `H18 =SUM(H8:H17)`
  - `I18 =SUM(I8:I17)`
  - `J18 =SUM(J8:J17)`
  - `K18 ="合計時間："&J18&" h"`

### 5. 不要シート整理
- `Link` と運用シートのみ残す。
- その他は削除（履歴保持が必要な場合は `.bak` を先に作成）。

### 6. 伝票PDFの提出用レイアウト
1. `guardman.xlsx` の `work_date` で `source_page` を日付昇順に並べる。
2. `ガードマン伝票.pdf` を A4縦 2-up（上下2枚）で再配置。
3. 出力例: `ガードマン伝票_日付昇順_2up.pdf`

### 7. 最終確認
- 日付順、人数列（昼/夜）、合計時間、氏名の重複/欠落を目視確認。
- 文字化けや時刻跨ぎ（21:00→02:00）を重点確認。
