# 舗装工事

会話数: 7 | パターン数: 5

## 1. 施工影響範囲に基づく広報対象リストの作成

道路使用許可や規制図に基づき、通行止めや工事の影響を受ける周辺住民・施設を特定し、広報活動用のリスト（Excel/CSV）を整備する。

### 手順

1. 規制図（PDF）を確認し、通行止め等の影響範囲を特定する
2. Googleマップや住宅地図情報を参照し、範囲内のマンション、公共施設、店舗を抽出する
3. 大きなマンションについては戸数や階数などの詳細情報を調査する
4. 抽出したデータを整理し、広報優先度や住所を含むExcelリストを作成する

出現頻度: 4回

## 2. 工事仕様書・管理基準の自動検索と照査

膨大な共通仕様書や施工管理の手引きの中から、特定の工種（区画線復旧、交通開放条件など）に関する基準を検索・抽出する。

### 手順

1. ドライブ内の仕様書フォルダ（道路編、共通編、総則等）をスキャンする
2. 特定のキーワード（例：「交通開放」「区画線」）でPDF内のテキストを検索する
3. 該当する管理基準や施工上の留意事項を抽出し、施工計画への適合性を確認する

出現頻度: 1回

## 3. 産廃集計データの整合性調整と報告書作成

産業廃棄物（アスファルト殻等）の運搬データを集計し、過積載防止などの法規制を遵守した形で報告書を再構成する。

### 手順

1. 既存の産廃集計表（Excel）を読み込む
2. 使用車両の最大積載量を調査し、実数値と比較する
3. 過積載となっているデータがある場合、規定内に収まるよう数値を調整（書き換え）する
4. 調整後のデータに基づき、提出用の報告書を更新する

出現頻度: 1回

## 4. 契約・設計図書のクロスチェック（整合性確認）

入札時の設計書、契約書、工事費内訳書の間で金額や数量、工期などの情報に矛盾がないか確認する。

### 手順

1. 入札、契約、着工時の各フォルダから関連PDFを収集する
2. PDFを画像化またはテキスト化し、主要な項目（直接工事費、諸経費、現場名など）を抽出する
3. 各書類間での数値の不一致や転記ミスがないか比較検証する

出現頻度: 1回

## 5. 地図データ(KML/HTML)による施工情報の可視化

広報リストや施工箇所情報をKML形式やHTML（Folium等）に変換し、視覚的なマップを作成する。

### 手順

1. Excel等の住所情報から緯度経度を取得または推測する
2. Pythonスクリプトを用いてKMLファイルを生成する
3. Webブラウザで閲覧可能なインタラクティブマップ（HTML）を出力し、関係者間で共有可能にする

出現頻度: 2回
