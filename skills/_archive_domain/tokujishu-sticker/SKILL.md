# 特定自主検査標章（特自ステッカー）

## 標章の読み方

三角形のステッカー。3辺に月(1～12)、頂点に年が記載。

```
      [年]        ← 頂点の数字 = 検査実施年（和暦・令和）
     /    \
   [月] ─ [月]    ← 3辺に1～12の月タブ
```

### 各要素の意味

| 要素 | 位置 | 意味 |
|------|------|------|
| 年 | 三角形の頂点 | 検査**実施**年（**和暦**。令和元年=2019年） |
| 月タブ | 三角形の3辺 | 切り取られた月 = 検査**実施**月 |
| 「済」マーク | 中央の赤地 | 検査済みを示す |
| No. | 中段 | 標章の管理番号（検査番号） |
| 検査員名 | 中段 | 検査を実施した検査員 |
| 検査業者名 | 下段 | 検査を実施した業者（登録番号付き） |

### 有効期限の算出

```
有効期限 = 検査実施年月 + 1年
```

例: 頂点「7」(令和7年=2025年)、月タブ切取「6」→ 2025年6月実施 → **2026年6月**まで有効

### 和暦→西暦変換

```
西暦 = 令和の年 + 2018
```

| 令和 | 西暦 |
|------|------|
| 5 | 2023 |
| 6 | 2024 |
| 7 | 2025 |
| 8 | 2026 |
| 9 | 2027 |

## 注意事項

- **頂点の年は和暦**。「7」を西暦2027年と読み間違えるな
- 標章のNo.は測点番号ではない。ノーマライザでstationに入れるな
- `photoCategory == "その他"` の写真からは `station_hint_from_result` をスキップする（normalizer/mod.rs で対応済み）
- クボタ等の一部メーカーは三角形ではなく四角形のステッカーで、「第1回検査は○年○月です」と明記されている場合がある。この場合はその日付がそのまま期限

## 期限チェックの手順

1. 機械フォルダのresult.jsonから `特定自主検査` を含むエントリを抽出
2. detectedTextから標章番号を取得
3. **写真を目視**で頂点の年（和暦）と切取月を確認（OCRでは月タブの切取位置を正確に読めないことが多い）
4. 和暦→西暦変換し、+1年で有効期限を算出
5. 工期と比較して期限切れリスクを判定
