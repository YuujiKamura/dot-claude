# AI & Machine Learning

## 1. Reference-Driven Safety Council Documentation

Japanese construction safety documents (災害防止協議会) must align with specific 'Model Regulations' (規定例) provided by the client. Practitioners prioritize extracting structural schemas from project-specific PDFs over generic parameter-based generation.

### Steps

1. Bypass parameter prompts by extracting the 'Constitution' or 'Regulation' structure from provided reference PDFs (規定例) to ensure project compliance.
2. Differentiate between 'Regulations' (organizational rules) and 'Annual Plans' (scheduling) to set the correct drafting tone and scope.
3. Identify organizational hierarchies (e.g., Chairman, Secretary roles) directly from provided PDF models instead of assuming generic industry structures.
